HOW IT WORKS
POWER ON SEQUENCE
Press BTN1 → NeoPixel blue sweep → OLED splash screen → system ready. Nothing works until BTN1 is pressed.
TWO MODES
MODE A — MANUAL
Place glass → LED green → press BTN2 → dispenser serves. All green-lit glasses are served in sequence.
MODE B — AUTO
Place glass → immediately starts serving. Extra glasses placed during serve go green (standby) and are served in queue order.
SERVE SEQUENCE
${['Glass placed → limit switch LOW → LED green','Serve triggered → servo rotates to slot angle (30°–180°)','Hose points at glass → pump ON','OLED shows live mL counter + progress bar','Pump OFF at target volume → servo home → LED red','Glass removed mid-serve → pump stops → LED orange (warning)'].map((s,i)=>`
${i+1}
${s}
`).join('')}
LED COLORS
${[['#4ade80','Green','Glass present, ready to serve'],['#60a5fa','Blue','Dispensing water'],['#f87171','Red','Glass full / done'],['#fb923c','Orange','Warning — glass removed'],['Off','Gray','No glass in slot'],['Boot','Cyan sweep','Power-on animation']].map(([col,name,desc])=>`
${name}
${desc}
`).join('')}
VOLUME CALIBRATION
The code uses time-based volume estimation: pump_time = target_mL ÷ ML_PER_SECOND. Default is 8.0 mL/s. To calibrate: run the pump for exactly 10 seconds, measure the water collected, divide by 10. Update ML_PER_SECOND in the code with your measured value.
